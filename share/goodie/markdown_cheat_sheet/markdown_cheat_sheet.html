Code Blocks 
    To produce a code block, indent every line of the block by at least 4 spaces or 1 tab.

Lists

Unordered lists use asterisks, pluses, and hyphens as list markers: Ordered lists use numbers followed by periods: 
*   Red
*   Green
*   Blue
