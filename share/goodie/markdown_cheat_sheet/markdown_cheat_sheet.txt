Markdown Syntax Cheat Sheet

See full syntax documentation at http://daringfireball.net/projects/markdown/syntax

Headers

# This is an H1
## This is an H2
###### This is an H6

Links

This is [an example](http://example.com/ "Title") inline link.
[This link](http://example.net/) has no title attribute.
This is a [relative](/relative/) link.
This is [an example][id] reference-style link.

Images

![Alt text](/path/to/img.jpg)

Styling

Emphasis: *em* or _em_

Strong: **strong** or __strong__

Blockquotes:
> This is the first level of quoting.
>
> > This is nested blockquote.
>
> Back to the first level.

Code Blocks
    To produce a code block, indent every line of the block by at least 4 spaces or 1 tab.

Lists
Unordered lists use asterisks, pluses, and hyphens as list markers:

*   Red
*   Green
*   Blue

Ordered lists use numbers followed by periods:

1.  Bird
2.  McHale
3.  Parish
